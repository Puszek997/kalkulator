# kalkulator

## Giga Kalkulator — Zaawansowany Kalkulator Naukowy / Advanced Scientific Calculator

A feature-rich C# console calculator application supporting a wide range of mathematical operations.

### Funkcje / Features

- **Podstawowe działania / Basic arithmetic**: addition, subtraction, multiplication, division, modulo, power
- **Zaawansowane / Advanced**: square root, nth root, absolute value, factorial, percentage
- **Trygonometria / Trigonometry**: sin, cos, tan, asin, acos, atan (degrees)
- **Logarytmy / Logarithms**: log₁₀, ln, logₙ, e^x
- **Pamięć / Memory**: MS, MR, MC, M+, M−
- **Historia / History**: view and clear calculation history
- **ANS**: use last result as input by typing `ANS`

### Uruchamianie / Running

```bash
dotnet run --project Kalkulator
```

### Testy / Tests

```bash
dotnet test
```
